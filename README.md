# Predictive Modeling Using Machine Learning

## Objective
Build a classification model using supervised learning techniques and perform Exploratory Data Analysis (EDA) on the Iris dataset to identify patterns, correlations, and important features affecting prediction.

---

# Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Seaborn
- Matplotlib

---

# Dataset Used
Iris Flower Dataset

The dataset contains flower measurements for:
- Setosa
- Versicolor
- Virginica

Features:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

---

# Exploratory Data Analysis (EDA)
The following analyses and visualizations were performed:

- Data preprocessing
- Missing value checking
- Duplicate value checking
- Statistical summary analysis
- Correlation heatmap
- Pairplot visualization
- Histograms
- Boxplots
- Scatter plots
- KDE distribution plots

---

# Model Used
Random Forest Classifier

---

# Steps Performed

1. Imported required libraries  
2. Loaded Iris dataset  
3. Performed data preprocessing  
4. Conducted Exploratory Data Analysis (EDA)  
5. Encoded target labels  
6. Split dataset into training and testing sets  
7. Trained Random Forest model  
8. Predicted test data  
9. Evaluated model using:
   - Accuracy Score
   - Classification Report
   - Confusion Matrix
10. Visualized feature importance

---

# Results
- Model achieved high accuracy on test data
- Petal-related features showed highest importance
- Correlation analysis revealed strong relationships between petal dimensions
- Setosa species was clearly separable from other classes
- Visualizations provided better understanding of feature distributions and relationships

---

# Feature Importance
Feature importance visualization was used to identify the most influential variables affecting flower classification.

Important features:
- Petal Length
- Petal Width

---

# Conclusion
This project demonstrates:
- Exploratory Data Analysis (EDA)
- Data visualization techniques
- Supervised machine learning
- Classification model building
- Model evaluation techniques

The Random Forest Classifier performed efficiently on the Iris dataset and achieved strong predictive performance with minimal overfitting.
